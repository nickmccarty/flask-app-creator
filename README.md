# Automated Creation of Flask App Scaffolding Using Self-Destructing Shell Scripts

> It's as easy as:

> - [x] Cloning the repo,
> - [x] Making ```flask_app_creator``` your current directory, and
> - [x] Running the ```create_flask_app``` shell script
